# Cahier des Charges Interactif

## 📋 Description

Application web permettant de créer, compléter, sauvegarder et exporter un cahier des charges de manière simple et guidée.

## 🚀 Fonctionnalités

- Assistant de saisie
- Barre de progression
- Sauvegarde locale
- Export JSON
- Import JSON
- Validation des champs obligatoires
- Interface responsive
- Compatible PC, tablette et smartphone

## 🎯 Objectifs

- Standardiser la collecte des besoins
- Réduire les oublis lors de l'analyse
- Faciliter les échanges entre commerce, technique et client
- Produire un document structuré

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript Vanilla
- GitHub Pages

## 📂 Utilisation

1. Ouvrir l'application
2. Compléter les différentes sections
3. Sauvegarder ou exporter le projet
4. Réimporter le fichier pour poursuivre la rédaction
